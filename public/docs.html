spee.ch

spee.ch is a single-serving site that reads and publishes images to and from the LBRY blockchain.

Help

Site Navigation

To publish a file, navigate to the homepage at spee.ch.

To view a claim, navigate to spee.ch/c/ + the name of the claim. E.g.spee.ch/doitlive.

API

To view a claim list, navigate to spee.ch/claim_list/ + the name of the claim. E.g.spee.ch/claim_list/doitlive. Note: this returns json
