spee.ch

spee.ch is a single-serving site that reads and publishes images to and from the LBRY blockchain.

Examples:

  • spee.ch/c/coconuts
  • spee.ch/c/test
  • spee.ch/c/doitlive

Help

Site Navigation

To publish a file, navigate to the homepage at spee.ch.

To view the first file at a claim, navigate to spee.ch/ + the name of the claim. E.g.spee.ch/doitlive.

To view the first file at a claim, navigate to spee.ch/ + the name of the claim + / + the claim_id. E.g.spee.ch/doitlive/c496c8c55ed79816fec39e36a78645aa4458edb5.

To view a batch of file at a claim, navigate to spee.ch/ + the name of the claim + / + all. E.g.spee.ch/doitlive/all.
